Comandos git

git init -> crea un repositorio en git
git add -> añade un archivo/s al area de pracicas de git
git commit  -m "nombre_identificativo"-> 
	añade al repositorio el/los archivo/s unicados
	en el área de practicas de git
git status -s -> devuelve un reporte del estado de los archivos
	del repositorio
git log --oneline -> devuelve una lista con las versiones 
	guardadas en el repositorio
git reset --hard codigo git -> restura el ducumento a la 
	versión capurada por git y almacenada en el
	repositorio con el código git
git push -> añade los cambios del archivo local a la nube
git pull -> añande los cambios del archivo nube al local
